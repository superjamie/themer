# Themer

## Usage

Create palette file which defines:

~~~
PAL_NAME
PAL_BACKGROUND
PAL_FOREGROUND
PAL_COLOR0 to PAL_COLOR15
~~~

Pass that as an argument to a themer:

~~~
bash theme-echo.sh pal-my-theme
~~~
